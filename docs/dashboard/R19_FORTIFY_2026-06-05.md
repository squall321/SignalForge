**3중 가드**:

2. `PRESERVE_EXISTING=true` 기본 → SELECT SQL 자체에서 `WHERE current_topic IS NULL OR current_topic = ''` 만 골라 옴, R18 식 "기존 분류 덮어쓰기" SQL 단계에서 차단
3. `RECLASSIFY_ALL=true AND ¬PRESERVE AND ¬DRY` 위험 조합은 `CONFIRM=I_KNOW` 미입력 시 `_validate_flags()` 에서 `sys.exit(3)`
