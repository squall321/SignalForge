3. **06-04 즉시 복귀**: 4,806건 (baseline) — burst 가 1일에 종료, 만성 폭증 없음.
4. **MEMORY 일치**: R6/R8 round 의 'HN +32k' / 'HN 점유율 0.3 → 22.67%' 기록과 정확히 매칭.

## 데이터 검증 (실측)

```
SELECT date_trunc('day', collected_at), COUNT(*)
FROM voc_records
WHERE collected_at > now() - interval '8 days'
GROUP BY 1 ORDER BY 1 DESC;
-- 2026-06-03 → 37,135 (확인)

SELECT date_trunc('year', published_at), COUNT(*)
FROM voc_records v JOIN platforms p ON p.id = v.platform_id
WHERE v.collected_at >= '2026-06-03' AND v.collected_at < '2026-06-04'
  AND p.code = 'hackernews'
GROUP BY 1 ORDER BY 1;
-- 2007 → 2026 전 분포 확인
```

판정: **NORMAL (의도된 HN 역사적 backfill)**.
